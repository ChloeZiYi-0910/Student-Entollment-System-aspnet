# ASP.NET Student Enrollment System

## Overview
The ASP.NET Student Enrollment System is a web-based application developed as a completed academic project.  
The system refines and improves the enrollment workflow inspired by the INTI International University Enrollment System, with a focus on structured backend design, data consistency, and system maintainability.

This project emphasizes backend logic, database integration, and role-based workflow management rather than UI-centric features.

---

## System Features
- Student enrollment and profile management
- Course registration and enrollment tracking
- Role-based access for students and administrators
- Structured data validation and backend processing
- Session-based workflow handling

---

## Tech Stack
- ASP.NET Core (Razor Pages)
- C#
- Entity Framework Core
- Microsoft SQL Server

---

## Project Scope
This system was developed as a school project and represents a complete functional implementation of an enrollment management workflow.  
The repository showcases system architecture, backend logic, and database interaction rather than visual interface design.
